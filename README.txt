Installation instructions
Two options exist to use intensify3D 
1)	For MATLAB users 
Download the MATLAB_Code.zip file and extract 
Go to the MATLAB_Code directory as the current working directory
 run file:  User_GUI_Intensify3D_1_2.m 
The GUI interface should open 
Follow instruction in Figure S1 as well as test data instructions.

2)	For standalone (PC only)
Download the Installer_web file and follow instruction
After installation, execute the Intensify3D.exe file
Follow instruction in Figure S1 as well as test data instructions.
